APOSTL Interactive Analysis

Once SAINTexpress has been run, APOSTL is able to read the resulting list.txt file. From here APOSTL calculates normalized spectral abundance factor (NSAF) values for each prey based on the average spectra observed for each bait. Additionally, APOSTL will calculate the NSAF Score as described below. Optionally, APOSTL calculates the probability of a specific interaction based on prey prevalence in the CRAPome.

1) List File

Please specify the List file output from SAINTexpress.

2) Prey

Please specify the Prey file used to generate the List file from SAINTexpress.

3) CRAPome

Optional: Please specify the output file from Workflow 1 from the CRAPome website. This can be generated by copying and pasting the first column of your Prey file into the Enter Protein Identifiers field and clicking the Download as tab-delimited file link under your Query Results .

4) Inter File

Please specify the Inter file used to generate the List file from SAINTexpress.

Once you have loaded your data into Galaxy, APOSTL will generate an HTML file with a link to your personal interactive analysis environment. To view this HTML file, click the icon on the new file in your history and open the link in a new browser tab

Within an interactive analysis environment, APOSTL offers various visualization and analysis options including:

1) Quality control: Correlations between replicate runs

2) Quality control: Boxplots of selected protein across all replicates

3) Bubble graphs for each bait with numerous customization options:

  • Axis and bubble scaling choices:
    • \(ln(NSAF) = ln(\frac{SpC/L}{\sum_{n=1}^N (SpC/L)})\)
    • \(\sum_{n=1}^N SpC\)
    • \(log_{2}(FC)\)
    • \(Saint Score\)
    • \(Log Odds Score\)
    • \(NSAF Score = ln(\frac{NSAF_T + ε}{\frac{NSAF_c} n + ε})\)

\(SpC\) = Spectral Count (or MS1 Intensity)

\(L\) = Amino Acid Length

\(FC\) = Fold Change

\(ε = 1/mean(\frac{SpC}{L})_c\)

\(n\) = # control replicates

  • Optional: bubble color corresponds to the CRAPome probability of a specific interaction in which an 80% cutoff is applied where prey with ≤ 80% are colored separately. Input is the respective output file from Workflow 1 from the CRAPome website.
  • Numerous cosmetic options including labels, color schemes and plotting themes

4) Density plots of all baits for options specified above

5) Cytoscape networks of all preys passing filtering criteria displaying newly identified bait-prey interactions

6) Filtered data displayed as a table

7) KEGG pathway analysis of filtered data

8) Gene Ontology (GO) term analysis of filtered data

All graphs, images and data can be saved using the Download buttons to generate 600 dpi figures or alternatively by RMB + Save As.

APOSTL Walkthrough

This Walkthrough will guide you through using APOSTL and the Galaxy framework it was built on.

Create User Account

In order to get full access to the all aspects of the server you must create an account.

  • To do so please select the Register button under the User tab on the menu.
  • Fill out the form with your e-mail, password and diplay name.
  • Once you click submit you will be redirected to the Home page.

APOSTL Tools

All of the tools in the APOSTL package are on the left located under APOSTL Tools.

Instructions for tools can be found at: SAINT User Manual and APOSTL Manual.

  • The DotPlot tool is a limited version of the full DotPlot tool that runs only the biclustering method of analysis. The full version can be found at Prohits Visualization Tool.

Workflows

Access the Published Workflows for both Scaffold and MaxQuant by selecting Shared Workflows under the Shared Data tab on the menu.

The MaxQuant and Scaffold Workflows are the same except the Preprocessing tool has MaxQuant of Scaffold selected respectively.

Select either Workflow by clicking on it.

  • The shared Worflow will show you all the steps involved, the Workflow can be imported with plus button, and downloaded using the floppy disk icon in the top right corner.

Once imported the Workflows can now be accessed from the Workflow menu tab.

The Workflow can be run or edited by clicking on the Workflow and selecting Run or Edit rexpectively.

When Run the Worklow steps will have certian variables preset but the Scaffold or MaxQuant input and bait file must be provided. Other variables can be used but are not needed (i.e. a custom fasta database for identification of proteins).

Once the job is set press the Run Workflow button at the bottom of the page.

Histories

Access the Published Histories for both Scaffold and MaxQuant by selecting Shared Histories under the Shared Data tab on the menu.

Select either History by clicking on it.

  • The shared History will show you all the steps completed, the History can be imported with the Import History in the upper right hand corner.

Once imported the History will be shown in the History right side column.

  • When clicked on the dataset will expand. The eye button will open the dataset for viewing in the web browser, and the floppy dick icon can be used to download the dataset.

The shiny version can be accessed by selecting the Shiny dataset, hitting the eye button and click on the APOSTL Interactive Analysis link (or right click and open in new tab).

Any job can be run with the same settings using the circular arrows button.
